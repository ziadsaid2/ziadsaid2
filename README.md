I work on **SaaS products**, **CRM systems**, and **messaging platforms** — mostly fixing the hard problems, untangling integrations between services, and shipping features across frontend and backend.

A lot of my time goes into debugging production issues, watching how systems actually behave with tools like **Datadog**, and sorting out things like webhooks, message flows, syncing, and real-time updates when they break or need to scale. I also work on existing products — adding features, refining UI/UX, and making the whole experience feel a bit sharper after each pass.

I'm solution-oriented and persistent. I like digging into problems until I understand how the systems involved actually talk to each other, then finding a fix that holds. I use AI as part of my workflow to move faster, but the engineering decisions stay mine.

> _I don't always have a solution, but I know how to find one._
